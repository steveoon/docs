## 什么是花卷智能体？

花卷智能体（HuaJune Agent）是专为**蓝领智能招募**打造的 AI 对话服务平台。通过简单的 RESTful API，你可以快速为招聘业务集成强大的 AI 能力，实现候选人沟通自动化、招聘流程智能化。

### 🎯 为招聘场景深度定制

花卷内置了蓝领招聘业务所需的各类工具，开箱即用：

<CardGroup cols={2}>
  <Card title="预约面试" icon="calendar-check">
    自动安排面试时间，发送提醒通知
  </Card>
  <Card title="岗位信息查询" icon="briefcase">
    智能解答岗位相关问题，提供详细信息
  </Card>
  <Card title="订单查询" icon="file-invoice">
    查询招聘订单状态和进度
  </Card>
  <Card title="订单分析" icon="chart-bar">
    分析招聘数据，生成业务洞察
  </Card>
</CardGroup>

### 💬 智能回复，零配置上手

**最大亮点**：无需任何额外配置，直接调用 API 即可获得针对蓝领招聘场景优化的智能回复。

系统自动理解候选人的咨询意图，并生成专业回复：

- 📋 **岗位咨询**：工作内容、任职要求、发展空间
- 💰 **薪资说明**：薪资结构、福利待遇、绩效奖金
- 🕐 **排班解释**：工作时间、班次安排、休息制度
- 📍 **地点介绍**：工作地点、交通路线、住宿安排
- 🤖 **7×24 响应**：随时为候选人提供即时、准确的回复

<Tip>
  开发者只需传入候选人消息，花卷会自动调用合适的工具、匹配最佳回复策略，无需手动配置业务规则。
</Tip>

  <Accordion title="智能对话" icon="comments">
