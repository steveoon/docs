## 认证机制

花卷智能体 API 使用 **Bearer Token** 认证机制。每个 API 请求都需要在 HTTP 请求头中携带有效的 API Key。

### 基本用法

在每个 API 请求的 `Authorization` 头中添加你的 API Key：

```http
Authorization: Bearer YOUR_API_KEY
```

<Warning>
  所有 API 请求都必须通过 HTTPS 协议发送，以确保 API Key 的安全传输。
</Warning>

## 获取 API Key

<Steps>
  <Step title="访问 Wolian AI 平台">
    前往 [客户端管理页面](https://wolian.cc/platform/clients-management)
  </Step>

  <Step title="登录账户">
    使用你的账户登录（与 `huajune.duliday.com` 使用相同的账户系统）
  </Step>

  <Step title="创建客户端密钥">
    点击页面中的 **"+ 创建"** 按钮，创建新的客户端和 API 密钥
  </Step>

  <Step title="激活密钥">
    创建后，确保密钥状态显示为 **"已激活"**（绿色标识）

    <Note>只有已激活的密钥才能用于 API 调用</Note>
  </Step>

  <Step title="复制密钥">
    点击密钥旁边的复制按钮，复制完整的 API Key 并妥善保存

    <Warning>密钥仅在创建时完整显示一次，请务必安全保存</Warning>
  </Step>
</Steps>

### API Key 格式

API Key 通常为 32 位字符串，格式如下：

```
31ad14.******************
fabbb5.******************
20bd38.******************
```

前 6 位为密钥标识符，后续为加密字符串。
